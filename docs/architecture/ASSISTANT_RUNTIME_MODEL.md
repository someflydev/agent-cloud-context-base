# Assistant Runtime Model

| Subsystem | Files involved | Command surface | Failure modes | Escalation rules |
| --- | --- | --- | --- | --- |
| Routers | `context/router/*.md`, `context/router/*.json`, `context/router/*.yaml` | `python3 scripts/work.py route-check "<prompt>"` when enabled | Wrong provider/runtime classification, over-broad context loading | Load only the next router or ask for the missing profile dimension. |
| Scenarios | `context/scenarios/*.md`, `context/scenarios/scenario-profile-map.yaml`, `context/router/scenario-router.md` | `context/workflows/plan-scenario-derived-repo.md` | Workload prompt maps to a deferred or unsupported cell | State the closest supported profile and the deferred gap. |
| Manifests | `manifests/*.yaml`, `manifests/README.md` | `python3 scripts/validate_manifests.py [name]` | Missing context refs, invalid isolation declarations, reference-only manifest used for generation | Fix manifest metadata before generation; do not scaffold reference-only profiles. |
| Profile composer | `scripts/accb_payload.py`, `context/accb/profile-rules.json`, `context/specs/`, `context/validation/` | `python3 scripts/accb_payload.py ... --output-dir <dir>` | Capability coverage gaps, unknown manifest, mismatched selection | Correct the selected archetype/stack/manifest tuple and rerun. |
| Validation gates | `scripts/validate_context.py`, `scripts/validate_iac_isolation.py`, `scripts/run_verification.py`, `verification/` | `python3 scripts/run_verification.py --tier fast|medium|local-provider|real-cloud|full` | Missing proof, environment-gated checks skipped, Docker/cloud credentials unavailable | Use `incomplete` when proof remains local-only; use `blocked` when required tooling or credentials are absent. |
| Runtime continuity | `scripts/work.py`, `context/*.example.md`, `PLAN.md`, `context/TASK.md`, `context/SESSION.md`, `context/MEMORY.md` | `resume`, `checkpoint`, `pause`, `done`, `next`, `verify` | Stale runtime notes, unreviewed changed files, claiming `done` without proof | Run `resume`, update notes, then validate before completion. |
| Memory | `memory/INDEX.md`, `memory/concepts/*.md`, gitignored `memory/sessions/` and `memory/summaries/` | `python3 scripts/work.py checkpoint` | Concept files stale after shipped prompts, index missing artifacts | Update concept wording and `memory/INDEX.md`; keep gitignored files out of the index. |
| Examples | `examples/catalog.json`, `verification/example_registry.yaml`, `examples/canonical-*` | `python3 scripts/verify_examples.py`, parity runners | Catalog/registry drift, tier status overwritten, local-provider or real-cloud gates run accidentally | Update only the tier actually run; keep billable lanes operator-gated. |
