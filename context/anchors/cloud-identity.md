# Cloud Identity Anchor

- Declare provider before loading provider-specific context.
- Declare runtime tier before loading stack packs.
- Declare primary language before loading stack packs.
- Declare IaC tool before loading IaC stack packs.
- Use Terraform or one Pulumi language for a non-comparative task.
- Declare dev/test isolation surface before generating any cloud resource.
- Keep workload identity separate between dev and test.
- Keep secret paths separate between dev and test.
