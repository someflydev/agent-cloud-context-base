# Function Stack Surface

PROMPT_09 establishes the accb function stack surface with nine stack documents: `aws-lambda-python.md`, `aws-lambda-typescript-node.md`, `aws-lambda-go.md`, `gcp-cloudfn-python.md`, `gcp-cloudfn-typescript-node.md`, `gcp-cloudfn-go.md`, `azure-fn-python.md`, `azure-fn-typescript-node.md`, and `azure-fn-dotnet-isolated.md`. These cover the primary AWS Lambda, GCP Cloud Functions Gen2, and Azure Functions language/runtime combinations; PROMPT_10 adds the managed-container stack surface.
