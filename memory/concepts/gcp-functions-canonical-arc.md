# GCP Cloud Functions Canonical Arc

PROMPT_19 seeded the canonical GCP Cloud Functions family with three
first-class examples: Python GCS OCR to Firestore, TypeScript HTTP Stripe
webhook handling, and Go Eventarc Monitoring incident routing. Each seed
includes Terraform plus matching Pulumi-language IaC, disjoint dev/test
isolation, smoke and replay tests, and minisky plus real-cost integration
lanes; PROMPT_20 mirrors this seed pass for Azure Functions.
