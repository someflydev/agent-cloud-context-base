# AWS Lambda Canonical Arc

PROMPT_18 seeded the canonical AWS Lambda family with three first-class examples:
Python S3 image moderation, TypeScript API Gateway Stripe webhook handling, and
Go SQS document translation. PROMPT_25 expands the surface to nine examples by
adding TypeScript and Go S3 moderation variants, a Python Stripe webhook, a
Python SQS translation variant, a TypeScript EventBridge CDC relay, and a
Python Cognito post-confirmation workflow. Each example includes Terraform plus
matching Pulumi-language IaC, disjoint dev/test isolation, smoke and replay
tests, and ministack plus real-cost integration lanes; PROMPT_26 completes the
GCP Cloud Functions expansion.
