# Container Apps Canonical Arc

PROMPT_22 completes the managed-container seed arc with two Azure Container Apps examples: `examples/canonical-container-apps/public-api-private-worker-jobs/python-fastapi/` for a public FastAPI app, internal worker, Service Bus-triggered Container Apps Job, Cosmos DB, Blob Storage, Key Vault, managed identity, and KEDA queue-depth scaling; and `examples/canonical-container-apps/dapr-pubsub-binding/dotnet-aspnet/` for two .NET 8 Container Apps using Dapr pub/sub, state, and secret components over Service Bus, Cosmos DB, and Key Vault. These examples establish the Azure `miniblue` Lane A local-provider pattern for managed containers and isolated Azure Lane B shape; PROMPT_23 begins the Kubernetes seed arc.
