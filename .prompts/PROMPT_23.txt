      - `teardown.sh` (`kind delete cluster`)

Lane B (ephemeral real):
  - `tests/integration/lane-b-ephemeral-real/` with `up.sh`,
    `test_platform.sh`, `down.sh`. Cluster lifetime is bounded by the
    suite; teardown is mandatory.
  - Cost band: significantly higher than function/container examples.
    Document the per-run cost band in each README ($5–$50 typical).
  - This lane requires real cloud credentials; CI default-skips.
